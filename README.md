Privacy Policy

Last updated: January 2026

1. Introduction

Welcome to PostingAPI (“we”, “our”, or “us”). Your privacy is important to us. This Privacy Policy explains how information is collected, used, and protected when you use our application.

PostingAPI is a desktop application built using Python (Tkinter). It does not operate as a public website or web-based platform. The app uses Meta (Facebook) APIs solely to publish content on Facebook Pages owned or managed by the developer.

By using this application, you agree to the collection and use of information in accordance with this Privacy Policy.

2. Information We Collect

We only collect information that is strictly necessary for the app to function.

a. Information Provided via Meta

Facebook Page IDs, names, and metadata for public Facebook Pages

Access tokens provided by Meta for authentication and API access

b. Content Data

Post content generated by the user for publishing (e.g., “Man of the Match” posts)

We do not collect:

Facebook passwords

Private messages

Non-public user data

Sensitive personal information

3. How We Use Information

Collected information is used only to:

Authenticate the app with Meta (Facebook)

Publish “Man of the Match” posts to a Facebook Page owned by the developer

Tag publicly available Facebook Pages, including:

A broadcast company’s Facebook Page (developer-owned), and

A player’s professional Facebook Page

Ensure correct and accurate Page tagging

Comply with Meta Platform Policies and applicable laws

The app does not analyze private user behavior or access private content.

4. Data Sharing

We do not sell, rent, or share user data with third parties.

Data may only be shared:

With Meta Platforms, Inc., as required to provide API functionality

When required by law or valid legal requests

5. Data Storage and Security

Data is accessed only when required to publish posts

No unnecessary data is stored

Reasonable technical and organizational measures are used to protect access tokens and app data

6. Data Retention

We retain data only for as long as necessary to provide app functionality or comply with legal requirements.

Users may revoke permissions or request deletion of their data at any time.
